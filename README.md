# Minesweeper

A classic Minesweeper clone built using **C#** and **WinForms**.

## ðŸŽ® How to Play
The goal of the game is to clear a rectangular board containing hidden "mines" without detonating any of them.

1. **Left Click**: Reveal a cell. If it's a mine, you lose!
2. **Right Click**: Place a flag on a cell you suspect contains a mine.
3. **Numbers**: The number on a revealed cell shows how many mines are adjacent to it.
4. **Win**: Reveal all safe cells to win the game.
